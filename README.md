# Persona Reliquary

A sculptural vessel for an enduring voice. The Persona Reliquary is a physical embodiment of a digital persona, presented as a classical bust and housed beneath an inverted glass cloche.

## Project Page

This project has its own GitHub Pages site with a Matrix chat widget for talking with the project faculty lead.

## Chat Widget

The project page includes a Matrix chat widget that:

- Allows direct conversation with the project faculty lead
